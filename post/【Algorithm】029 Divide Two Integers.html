31. 下一个排列数

题目表述

在不使用 * / %操作符的情况下，计算两个数的商
The integer division should truncate toward zero, which means losing its fractional part. For example, 8.345 would be truncated to 8, and -2.7335 would be truncated to -2.

Note: Assume we are dealing with an environment that could only store integers within the 32-bit signed integer range: [−2^31, 2^31 − 1]. For this problem, if the quotient is strictly greater than 231 - 1, then return 2^31 - 1, and if the quotient is strictly less than -2^31, then return -2^31.

truncate: 截断

分析

翻倍来测试大概范围

例如：60 / 8 = (60-32)/8 + 4 = (60- 32 - 16)/8 + 4 + 2 = 1+4+2=7

dividend=60 , divisor=8 。

若(temp_divisor + temp_divisor) < dividend，

count =count+count;

temp_divisor = temp_divisor + temp_divisor

若此时(divisor + divisor) >= dividend, 回溯return Backtracking(count, dividend-divisor, divisor)
