int (*(*v)[])(); 这种复杂的类型如何判断?

例：int (*v)[5];
1、先看变量名: *v v是一个指针
2、向右看，看它是什么: [5] 指向数组
3、再向左看，看它是什么类型: int [5] 指向int型的数组
4、不要跨过括号去看
