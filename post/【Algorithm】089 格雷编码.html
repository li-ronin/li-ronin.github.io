89.格雷编码

题目表述

n 位格雷码序列 是一个由 2n 个整数组成的序列，其中：

  • 每个整数都在范围 [0, 2n - 1] 内（含 0 和 2n - 1）
  • 第一个整数是 0
  • 一个整数在序列中出现 不超过一次
  • 每对 相邻 整数的二进制表示 恰好一位不同 ，且
  • 第一个 和 最后一个 整数的二进制表示 恰好一位不同

给你一个整数 n ，返回任一有效的 n 位格雷码序列 。

分析

G(n+1) = G(n) 并上 R(n)

R(n) = G(n)逆序过来然后在前一位加个一。

例如n=2，G(2) = [00, 01, 10, 11] ，G(2)的逆序 [11, 10, 01, 00]，在前一位加个一R(n) = [111, 110, 101, 100]

G(3) = G(2)∪R(2) =[000, 001, 010, 011, 111, 110, 101, 100 ]
